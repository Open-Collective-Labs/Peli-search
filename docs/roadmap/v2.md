# v2 — Vector and Hybrid Search

Focus: Semantic search capabilities on top of the v1 foundation.

## Features

### Vector Search
- Define vector fields in mappings (configurable dimensions)
- HNSW approximate nearest neighbor index
- Similarity metrics: cosine, dot product, euclidean
- Vector quantization (f16, i8 scalar)
- k-NN search queries
- Filtered vector search (post-filter and pre-filter)

### Hybrid Search
- Combined BM25 + vector search queries
- Reciprocal Rank Fusion (RRF) for result merging
- Configurable RRF parameters (k, per-method weights)
- Normalized score ranges

### Performance Targets
- Vector index throughput: 5K docs/sec (384-dim vectors)
- ANN recall @ 10: > 95%
- Hybrid query latency: < 100ms (P99) for 1M vectors

## Out of Scope for v2

- Replication / failover
- Distributed HNSW
- Online learning / feedback loops
